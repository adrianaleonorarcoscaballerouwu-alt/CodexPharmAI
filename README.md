Este software utiliza Inteligencia Artificial (Prototypical Networks)
para predecir la toxicidad de compuestos farmacéuticos mediante
descriptores moleculares normalizados.

INSTRUCCIONES RÁPIDAS DE USO:
1. Abre la carpeta llamada "Software_PharmAI".
2. Ejecuta el archivo llamado "app" (el que tiene el icono de aplicación).
3. Asegúrate de que la base de datos en formato Excel permanezca 
   en esa misma carpeta para el correcto análisis del modelo.
4. Los reportes generados pueden ser almacenados en la carpeta 
   "Resultados_y_Reportes".

¡Gracias por usar PharmAI para la investigación científica!
